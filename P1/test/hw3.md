##### 数据限制

- 本次作业减少了对于输入的限制，因此可能出现的情况包括但不限于：
  - 三角函数内部嵌套表达式因子：`sin((x*x))`
  - 自定义函数定义或最后一行待展开表达式中有多层括号：`f(x) = x*(1+x*(1+x))` `x * (x + x * (x + x))`
  - 自定义函数调用中允许出现表达式因子、求和函数或自定义函数调用 `f((x+1), sum(i, 1, 10, (i*x)), g(x))`；当 `f(x) = sin(x**2)` 时，也允许 `f((x+1))` 或 `f(x**2)` 这样的调用
